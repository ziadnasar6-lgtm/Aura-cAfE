# ☕ Aura Cafe | Premium Coffee Landing Page

A modern, elegant, and fully responsive single-page landing page designed for a premium coffee brand. Built with a sophisticated **Premium Dark Theme** featuring sleek glassmorphism effects, dynamic filtering, and smooth scroll animations.

## ✨ Features

*   **Premium Dark Aesthetics:** Designed with a rich color palette (`#0a0908`, `#151413`) accented by luxury gold highlights (`#d4a373`).
*   **Fully Responsive Layout:** Optimized seamlessly for desktops, tablets, and smartphones using modern CSS Media Queries.
*   **Dynamic Interactive Menu:** Filterable coffee and pastry categories using animated tabs.
*   **Modern Navbar UI:** Fixed glassmorphism header (`backdrop-filter`) that transforms smoothly upon scrolling.
*   **Immersive Hero Component:** Fixed background parallax effect with clean typographical gradients.
*   **Smooth Visual Animations:** Integrated with scroll-reveal frameworks via the `.reveal` styling architecture.
*   **Responsive Hamburger Menu:** Tailored overlay navigation drawer for mobile and compact viewports.

---

## 🛠️ Technologies Used

*   **HTML5:** Structured semantic elements (`<header>`, `<section>`, `<footer>`, `<nav>`) for optimal SEO and accessibility.
*   **CSS3:** Custom properties (CSS variables), CSS Grid, Flexbox, transitions, and hardware-accelerated transforms.
*   **Google Fonts:** Clean global typography leveraging the *Outfit* (headings) and *Inter* (body text) font families.
